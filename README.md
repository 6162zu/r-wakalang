Rのわからないことへの質問を日本語でする場所
「こういうのが欲しい」という雑な妄想や夢でもいいかも。
そうすると匿名知的集団ホクソエムが開発してくれるかも・・・？

### 何をするか？

- 知りたいことをissueで聞く
- あったらなーというものをissueに書く
- issue立てた人が納得したらcloseする
- 得られた成果をtips.md(まだない)等に書く

### もっとラフな雑談がしたい！
Gitterを使った[チャットルーム](https://gitter.im/TokyoR/r-wakalang)があります！
